# Neovim Config
## Features
- lsp (specific languages can be enabled or disabled in init.lua)
- treesitter
- completions
- lualine
- noice
- debugging (for ui nvim-dap-ui and nvim-dap-view both are installed. see mappings for toggling between them)
- persistence with resession nvim
- dash
- harpoon
- context management
- indent lines
- autopairs
- telescope
- neo-tree
- undotree
- gitui
- diagnostics navigation
## Plugins
All plugins are in 'plugins' directory and subdirectories. This config uses lazy for plugin management.
## Mappings
All mappings are defined 'mappings' directory. The modules are imported where the assignments or includes are required.
